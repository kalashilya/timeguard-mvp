# HW2. Пользователь, функции и user flow

## Пользователь

Занятый студент или специалист, которому нужно планировать день по времени.

## Основные функции MVP

- создание профиля;
- добавление задач;
- выбор даты;
- выбор времени начала и окончания;
- проверка пересечений;
- сортировка задач;
- кабинет;
- тарифный шаг.

## User flow

1. Открыть сайт.
2. Сохранить профиль.
3. Добавить первую задачу.
4. Добавить вторую задачу.
5. Проверить план дня.
6. Попробовать добавить конфликтную задачу.
7. Получить предупреждение.
8. Открыть кабинет.

## Критерий успеха

Пользователь за короткое время получает план дня и понимает, какие задачи конфликтуют.
