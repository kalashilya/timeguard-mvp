# TimeGuard Planner MVP

Веб-версия учебного MVP для планирования дня без конфликтов времени.

## Что делает TimeGuard

Пользователь добавляет задачи на дату, указывает время начала и окончания, категорию и приоритет. Перед сохранением сервис проверяет, не пересекается ли новая задача с уже существующими делами.

## Файлы проекта

- `app.html` — главная страница и планировщик;
- `app.css` — дизайн;
- `app.js` — логика профиля, задач, конфликтов, лимитов и тарифов;
- `cabinet.html` — кабинет;
- `pricing.html` — тарифы;
- `payment-success.html` — учебная активация тарифа;
- `admin.html` — учебная admin-сводка.

## Данные

Используется браузерное хранилище:

- `timeguard_profile_v1` — профиль пользователя;
- `timeguard_tasks_v1` — задачи;
- `timeguard_stats_v1` — статистика конфликтов.

## Роли

- `guest` — посетитель без профиля;
- `user` — пользователь с профилем и сохранёнными планами;
- `admin` — учебная роль для просмотра сводки MVP.

## Ограничение MVP

Реальной серверной авторизации нет. Это учебный MVP, который показывает пользовательский сценарий и продуктовую логику. Следующий этап развития — перенос данных в Supabase или Firebase.
