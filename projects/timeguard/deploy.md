# Deploy TimeGuard Planner

## Платформа публикации

Рекомендуется GitHub Pages, потому что проект является статическим сайтом на HTML, CSS и JavaScript.

## Шаги публикации

1. Открыть настройки репозитория.
2. Перейти в раздел Pages.
3. Выбрать Deploy from a branch.
4. Branch: main.
5. Folder: root.
6. Сохранить настройки.

## Проверка после публикации

- главная страница открывается;
- кнопка ведёт в приложение;
- профиль сохраняется;
- задачи добавляются;
- конфликт времени блокируется;
- кабинет работает;
- тарифы открываются;
- учебный тариф активируется;
- admin-страница показывает ограничение доступа без роли admin.

## Как объяснять деплой

TimeGuard опубликован как статический frontend на GitHub Pages. Для первой версии MVP это достаточно: проект показывает основной пользовательский сценарий и хранит данные в браузере через localStorage.
