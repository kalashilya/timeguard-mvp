# Шпаргалка для защиты TimeGuard Planner

## Что это за проект

TimeGuard Planner — учебный MVP планировщика дня. Он помогает пользователю добавлять задачи на конкретную дату и заранее видеть, если дела пересекаются по времени.

## Проблема

Обычные списки дел показывают только перечень задач, но не проверяют реалистичность расписания. Пользователь может поставить две задачи на одно время и заметить это слишком поздно.

## Пользователь

Студент, молодой специалист или менеджер, которому нужно совмещать учёбу, работу, встречи и личные дела.

## Ценность

Пользователь быстрее собирает реалистичный план дня и заранее видит, какие дела конфликтуют по времени.

## Основной сценарий

1. Открыть сайт.
2. Создать учебный профиль.
3. Добавить задачи на дату.
4. Показать сортировку по времени.
5. Попробовать добавить задачу с пересечением.
6. Показать предупреждение.
7. Открыть кабинет.
8. Показать тарифный шаг.

## Данные

- профиль пользователя;
- роль;
- тариф;
- дата задачи;
- название;
- время начала;
- время окончания;
- приоритет;
- категория;
- комментарий;
- статус выполнения.

## Хранение

В учебной версии данные хранятся в браузере через localStorage. Для настоящего продукта нужен backend и база данных.

## Роли

- guest — посетитель без профиля;
- user — пользователь с профилем;
- admin — учебная роль для просмотра сводки MVP.

## Paywall

Free-план ограничивает количество задач и горизонт планирования. Когда пользователь достигает лимита, появляется тарифное окно. Тарифная часть учебная, без настоящих платежей.

## Безопасность

- проверка обязательных полей;
- проверка времени начала и окончания;
- проверка пересечений;
- чекбокс «Я не робот»;
- cooldown между отправками формы;
- ограничение admin-страницы по роли.

## Что делал ИИ

ИИ помогал сформулировать идею, структуру MVP, пользовательский сценарий, код и документацию. Итоговый сценарий проверяется вручную через сайт.

## Что улучшить после MVP

- подключить Supabase или Firebase;
- добавить настоящую регистрацию;
- сделать недельный календарь;
- добавить уведомления;
- добавить совместные планы;
- подключить аналитику;
- перенести проверку ролей и тарифов на сервер.

## Речь на 40 секунд

Я сделал TimeGuard Planner — учебный MVP планировщика дня. Проблема в том, что обычные списки дел не проверяют, реально ли выполнить расписание. Пользователь может поставить две задачи на одно время. TimeGuard решает это: пользователь добавляет задачи с датой и временем, а система проверяет пересечения, сортирует задачи и не даёт сохранить конфликт. В проекте есть профиль, роли, кабинет, Free-лимит, учебный тарифный сценарий и базовые проверки безопасности. Данные в MVP хранятся в localStorage, а следующим этапом можно подключить Supabase и настоящую авторизацию.
