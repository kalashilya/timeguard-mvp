# Prompt 01. Idea generation

Задача: придумать учебный MVP для курса, который можно реализовать как статический сайт и показать на защите.

Ограничения:

- понятная проблема;
- один основной пользовательский сценарий;
- возможность показать данные;
- простая реализация на HTML, CSS и JavaScript;
- возможность развить проект после MVP.

Итоговая идея: TimeGuard Planner — планировщик дня, который предупреждает о пересечениях задач по времени.
