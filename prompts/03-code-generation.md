# Prompt 03 — Code generation

Задача: создать frontend MVP TimeGuard Planner на HTML, CSS и JavaScript.

Требования к интерфейсу:

- современный лендинг;
- форма профиля;
- форма добавления задачи;
- список задач на выбранную дату;
- предупреждение о конфликте;
- кабинет;
- тарифная страница;
- admin-страница.

Требования к логике:

- хранить данные в localStorage;
- проверять обязательные поля;
- проверять, что окончание позже начала;
- находить пересечения задач по времени;
- сортировать задачи по времени и приоритету;
- ограничивать Free-план;
- показывать тарифный шаг;
- учитывать роль admin.

Критерий готовности:

Пользователь должен пройти сценарий от создания профиля до конфликта времени и кабинета без ручной правки кода.
